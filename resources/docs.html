StyleDocco 0.3.0 default styles

body, a, code and pre are the only global styles we define, keep others under prefixed class names to reduce risk of conflicts.

body {
    margin: 0;
    font: 13px/1.6 sans-serif;
    background-color: #fafafa;

Using gradients for faux columns and checkerboard pattern. We use a checkerboard pattern to be able to see the edges of elements regardless of their color.

    background-image: -webkit-linear-gradient(left, transparent 56%, #eee 56%),
                      -webkit-linear-gradient(45deg, #fff 25%, transparent 25%, transparent 75%, #fff 75%, #fff),
                      -webkit-linear-gradient(45deg, #fff 25%, transparent 25%, transparent 75%, #fff 75%, #fff);
    background-image:    -moz-linear-gradient(left, transparent 56%, #eee 56%),
                         -moz-linear-gradient(45deg, #fff 25%, transparent 25%, transparent 75%, #fff 75%, #fff),
                         -moz-linear-gradient(45deg, #fff 25%, transparent 25%, transparent 75%, #fff 75%, #fff);
    background-image:      -o-linear-gradient(left, transparent 56%, #eee 56%),
                           -o-linear-gradient(45deg, #fff 25%, transparent 25%, transparent 75%, #fff 75%, #fff),
                           -o-linear-gradient(45deg, #fff 25%, transparent 25%, transparent 75%, #fff 75%, #fff);
    background-size: 100%, 20px 20px, 20px 20px;
    background-position: 0 0, 0 0, 10px 10px;
    background-attachment: fixed;
}
